## 🐤 Custom Flappy Bird Game (Object-Oriented)

This project is a fun and personalized remake of the classic Flappy Bird game built using Python and Pygame.

Unlike the original version, this game allows players to choose from multiple custom characters, each with unique visuals (including human face-based characters), adding a humorous and creative twist to the gameplay.

The game is developed using Object-Oriented Programming (OOP) principles, with separate classes for the Bird, Pipes, and Game logic, making the code modular and easy to understand.

### 🎯 Features
- Multiple character selection screen
- Smooth physics (gravity and jump mechanics)
- Dynamic pipe generation
- Collision detection system
- Score tracking system
- Game over screen with replay option
- Clean UI with custom fonts and graphics

### 🛠️ Built With
- Python
- Pygame
- Object-Oriented Programming (OOP)

### 🎮 How to Play
- Press **Space** or **Click** to make the character jump
- Avoid hitting the pipes
- Try to achieve the highest score!

This project was created as part of learning game development and applying OOP concepts in a practical way.
